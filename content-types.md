# Content Type Formulas — @findocere

## 7 Content Formulas

Reference: [[engine/hooks]] for hook selection. [[voice/brand-voice]] for tone. [[platforms/x]] for format rules.

### F1 — The Practitioner Confession

**Purpose:** Build trust through vulnerability + data. Findo's highest-engagement format.
**Structure:**
1. Hook (admission of mistake or emotion) — use H1 or H7
2. Context (what happened, with specific numbers)
3. Emotional layer (2 sentences on how it felt — PTSD, hesitation, frustration)
4. Lesson (one clear, reusable takeaway)
5. CTA (ask for shared experience — R1, R3, or R5)

**Format:** Single tweet with screenshot (JN) or short thread
**Audience:** [[audience/defi-practitioners]] primarily
**Engagement target:** Replies (people sharing their own experiences)
**Frequency:** 2x per week
**Tag rule:** Tag 0-2 accounts who'd relate

**Template:**
```
[Admission — what happened]

[Specific numbers — what the position did]

[Emotion — how it felt, reference to past experiences]

[Lesson — one reusable insight]

[Question — forces opinion, not agreement]

DYOR, NFA.
```

---

### F2 — The Counterintuitive Take

**Purpose:** Stop scrolls by challenging popular assumptions with evidence.
**Structure:**
1. Bold claim (contradicts popular belief) — use H2 or H8
2. Evidence (on-chain data, personal results, specific numbers)
3. Reframe (new mental model)
4. CTA (invite disagreement — R2)

**Format:** Single tweet or short thread
**Audience:** [[audience/defi-practitioners]] and [[audience/builders]]
**Engagement target:** Quote tweets (people reacting publicly)
**Frequency:** 1-2x per week

---

### F3 — The Technical Teardown

**Purpose:** Go deep on one mechanic. Establishes authority.
**Structure:**
1. Hook (H5 or H6) + 🧵
2. Quick primer (what the thing is, in 2 sentences)
3. Deep breakdown (3-4 tweets with one screenshot/visual per tweet)
4. "My exact setup" (personal configuration, real numbers)
5. Question (invite confusion admissions — R4)

**Format:** Thread (5-7 tweets)
**Audience:** [[audience/defi-practitioners]]
**Engagement target:** Bookmarks (dense reference value)
**Frequency:** 1x per week (Tuesday)

---

### F4 — The Hack Drop

**Purpose:** Single immediately actionable tactic. Maximum bookmark density.
**Structure:**
1. [What to do] → [Why it works] → [What you'll see]
2. That's it. One tweet. No thread.

**Format:** Single tweet (ST)
**Audience:** All segments
**Engagement target:** Bookmarks
**Frequency:** 2x per week

**Template:**
```
[Specific action to take]: [specific tool/setting/metric to use]. If [condition], then [consequence]. [Why this matters in one sentence].
```

---

### F5 — The Builder Log

**Purpose:** Document the MeteorWatcher agent build in public.
**Structure:**
1. Status update (what shipped/broke this week)
2. One specific surprise or lesson
3. What's next
4. CTA for builders (R4 or Q1)

**Format:** Single tweet or short thread
**Audience:** [[audience/builders]]
**Engagement target:** Replies from builder accounts
**Frequency:** 1x per week

---

### F6 — The Conviction Stack

**Purpose:** Present a thesis with layered evidence.
**Structure:**
1. Thesis statement (bold, specific)
2. Evidence layer 1 (data)
3. Evidence layer 2 (mechanism)
4. Evidence layer 3 (trend)
5. Personal stake (position disclosure)
6. CTA (invite counter-thesis — Q2)

**Format:** Thread (5-6 tweets)
**Audience:** [[audience/defi-practitioners]] + governance nerds
**Engagement target:** Quote tweets
**Frequency:** 1x every 2 weeks

---

### F7 — The Benchmark Post

**Purpose:** Raw performance data, honest assessment.
**Structure:**
1. Raw numbers (each position, net return)
2. Context (timeframe, market conditions)
3. Honest comparison (vs. simple holding, vs. benchmarks)
4. What went right / what went wrong
5. CTA (compare numbers — R1)

**Format:** Thread or single tweet with screenshot
**Audience:** [[audience/defi-practitioners]]
**Engagement target:** Replies (people sharing their own numbers)
**Frequency:** Weekly (Sunday watchlist) or monthly (full report)

## Formula Selection Logic

| Situation | Formula | Hook |
|---|---|---|
| Made a mistake / emotional trade | F1 | H1 |
| Data contradicts popular belief | F2 | H2 |
| Explaining how something works | F3 | H5 or H6 |
| Quick actionable tip | F4 | H3 or H4 |
| Agent build update | F5 | H4 |
| Governance thesis | F6 | H2 |
| Sharing performance data | F7 | H4 |
